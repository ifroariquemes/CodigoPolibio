## Como usar

```
$ polibio -h

---- Instituto Federal de Rondônia - Campus Ariquemes ----
Criptografa e decriptografa mensagens usando Código de Políbio
N. A. V. Simões <natanael.simoes@ifro.edu.br>
http://github.com/ifroariquemes/CodigoPolibio

Uso:

      polibio -c|-d mensagem

 -c mensagem      Criptografa uma mensagem
 -d mensagem      Decriptografa uma mensagem
 -h               Mostra essa mensagem

 Criptografar: polibio -c mensagem
 Decriptografar: polibio -d 3315344311221533
 ```
